# Stellar burger (Yandex practicum)
Бургерная в космосе

## Stack
React, Redux, TypeScript, Jest, Cypress

## Link demo
https://grimm1994.github.io/react-burger/

## Install packages
npm install

## Launch project
npm run start

## Build project
npm run build

## Jest test
npm run test

## Cypress test
npm run cypress

## Deploy to Github pages
npm run deploy
